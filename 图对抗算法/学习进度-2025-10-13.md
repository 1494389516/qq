   ```
   第1层：数据清洗
   ✓ 检测可疑购买（新注册、低信用、未实名）
   ✓ 移除100%的刷单记录
   → 基础防线
   
   第2层：对抗训练
   ✓ 混入刷单样本训练
   ✓ 降低可疑购买权重
   → 提升鲁棒性
   
   第3层：动态重训练
   ✓ 时间衰减（旧数据权重低）
   ✓ 快速更新（缩短攻击窗口）
   → 持续优化
   
   第4层：实时监控
   ✓ 检测异常增长（商品15：3→13次，增长4.3倍）
   ✓ 快速报警
   → 及时发现
   
   第5层：提高攻击成本（经济手段）← 核心！
   ✓ 强制实名认证
   ✓ 信用分门槛（>600）
   ✓ 注册时长要求（>30天）
   → 刷单成本：100元 → 500元（5倍）
   
   第6层：降低攻击收益（经济手段）← 核心！
   ✓ 可疑商品降权70%
   ✓ 延迟生效（24小时）
   ✓ 人工审核
   → 刷单收益：1000元 → 100元（降90%）
   
   第7层：法律威慑
   ✓ 10个账号进入黑名单
   ✓ 收集10条证据
   ✓ 配合公安打击
   → 心理威慑，不敢干
   ```
